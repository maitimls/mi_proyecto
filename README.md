# Análisis de las Ventas de una Cafetería

Este proyecto analiza un dataset de las ventas de una cafetería con el objetivo de analizar qué productos son los más deseados y buscar relaciones a través de la visualización de los resultados tras una previa limpieza del dataset.

## Estructura
- `/data`: dataset original y dataset limpio
- `/notebooks`: EDA: Descripción, Limpieza y Visualización

## Requerimientos
Python 3.13.7
Pandas, Seaborns, Matplotlib, Numpy.
